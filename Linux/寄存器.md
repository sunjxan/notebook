Intel 8086 Family Architecture

```
        General Purpose Registers 通用寄存器       
        8位    16位
        AH/AL  AX   Accumulator 累加            
        BH/BL  BX   Base 基础                  
        CH/CL  CX   Counter 计数              
        DH/DL  DX   Data 数据                  
                                        
        (ExX) indicates 386+ 32 bit register    
        (RxX) indicates 386+ 64 bit register
        (R8 ~ R15) indicates 386+ 64 bit register
        
        Segment Registers 段寄存器
        16位
        CS     Code Segment 代码段
        DS     Data Segment 数据段
        SS     Stack Segment 堆栈段
        ES     Extra Segment 附加段
        (FS)    386 and newer
        (GS)    386 and newer
   
        Pointer Registers 指针寄存器                      
        16位
        SI  Source Index 源指针                 
        DI  Destination Index 地址指针            
        IP  Instruction Pointer 指令指针

        Stack Registers 堆栈寄存器
        16位
        SP  Stack Pointer 堆栈指针
        BP  Base Pointer 基础指针
        
        (ESI EDI EIP ESP EBP) indicates 386+ 32 bit register    
        (RSI RDI RIP RSP RBP) indicates 386+ 64 bit register

        Status Registers 标志寄存器
        16位
        FLAGS
        
        (EFLAGS) indicates 386+ 32 bit register    

        Special Registers (386+ only)

        CR0     Control Register 0        DR0    Debug Register 0
        CR2     Control Register 2        DR1    Debug Register 1
        CR3     Control Register 3        DR2    Debug Register 2
                                          DR3    Debug Register 3
        TR4     Test Register 4           DR6    Debug Register 6
        TR5     Test Register 5           DR7    Debug Register 7
        TR6     Test Register 6
        TR7     Test Register 7
        
        Register          Default Segment    Valid Overrides

        BP                      SS              DS, ES, CS
        SI or DI                DS              ES, SS, CS
        DI strings              ES              None
        SI strings              DS              ES, SS, CS
```
